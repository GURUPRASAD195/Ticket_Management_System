# Ticket Management System

This is a full-stack Ticket Management System application built with a Java backend and a web frontend.

## Project Structure
* **`/src`**: Java Backend application source code.
* **`/ticket-frontend`**: Frontend user interface application.

## Getting Started

### Backend Setup
1. Navigate to the root directory.
2. Run the Java application using Maven:
   ```bash
   ./mvnw spring-boot:run
   ```

### Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd ticket-frontend
   ```
2. Install the necessary packages:
   ```bash
   npm install
   ```
3. Start the local development server:
   ```bash
   npm start
   ```
